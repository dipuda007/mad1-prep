Lesson 8: Flask Hello World

Goal: run a web server. This exact task ("write a route that prints hello world") is asked LIVE in vivas — practice until you can do it from memory in under a minute.

What is Flask?

A lightweight Python web framework: it receives HTTP requests, lets you map URLs to Python functions ("routes"), and sends back responses. "Micro" framework = gives you the core and stays out of the way (vs Django which bundles everything).

The smallest possible app

In your project folder create app.py:

from flask import Flask

app = Flask(__name__)          # create the application object

@app.route('/')                # when browser asks for "/"
def home():                    # ...run this function
    return 'Hello, world!'     # ...and send this back

if __name__ == '__main__':
    app.run(debug=True)        # start the server

Run it:

python app.py

Terminal shows Running on http://127.0.0.1:5000. Open that in your browser. Stop the server with Ctrl+C.

Line-by-line (memorize these answers)

LineWhat it does
Flask(__name__)creates the app; __name__ tells Flask where the file is so it can find templates/ and static/ folders
@app.route('/')decorator binding URL path "/" to the function below it
app.run(debug=True)starts the development server; debug mode auto-restarts on code changes and shows detailed error pages

More routes + dynamic routes

@app.route('/about')
def about():
    return 'This is the about page'

@app.route('/event/<int:event_id>')       # <int:event_id> captures a number from the URL
def show_event(event_id):
    return f'You asked for event number {event_id}'

@app.route('/course/<course_name>')      # captures text — viva favourite live task!
def course(course_name):
    return f'Course is {course_name}'

Visit /event/5 → "You asked for event number 5". The URL piece becomes a function argument.

Changing the port (asked in vivas!)

app.run(debug=True, port=8080)     # default port is 5000
Rapid-fire answers: default Flask port? 5000. Can it be changed? Yes, port= in app.run(). What is debug=True? auto-reload + error pages; must be off in production. Can a route return a boolean? No — returning True directly raises an error; wrap it: str(True), or return JSON/dict. Valid return types: string, rendered template, redirect, dict (becomes JSON), tuple with status code.
