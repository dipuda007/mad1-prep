Q&A: Python (38 questions)

Every one of these has been asked in a real MAD-1 viva or is one step away from one that has.

1. Difference between a list and a tuple?

List [1,2,3] is mutable — you can append, remove, change items. Tuple (1,2,3) is immutable — fixed after creation. Tuples are slightly faster and usable as dictionary keys.

2. What does mutable/immutable mean?

Mutable = can be changed in place after creation (list, dict, set). Immutable = cannot (int, float, str, tuple, bool). "Changing" a string actually creates a new string.

3. Difference between a list and an array?

A Python list holds items of any mixed types and resizes freely. An array (array module / NumPy) holds items of one fixed type, stored compactly, faster for math.

4. What is a dictionary?

Key→value mapping: {"name": "Music Night", "slots": 10}. Lookup by key, keys unique. request.form behaves like one.

5. List vs set?

List: ordered, allows duplicates, index access. Set: unordered, unique elements only, fast membership tests.

6. What is an f-string?

String with embedded expressions: f"Event {name} has {slots} slots" — the {} parts are evaluated and inserted.

7. == vs = ?

= assigns a value to a name; == compares two values and returns True/False.

8. What is a function? How do you define one?

A reusable named block taking inputs and optionally returning a value. def add(a, b): return a + b. Every Flask route is a function.

9. What is a lambda function?

A small anonymous one-expression function: square = lambda x: x*x. Used where a short throwaway function is needed, e.g. as a sort key: sorted(events, key=lambda t: t.name).

10. What is a decorator?

A function that wraps another function to add behaviour without editing it, applied with @. In my project @app.route('/login') is a decorator that registers the function as the handler for that URL.

11. What is a class? What is an object?

A class is a blueprint describing data (attributes) and behaviour (methods); an object/instance is one concrete item made from it. class Event(db.Model) is the blueprint; each event row I load is an object.

12. What is __init__?

The constructor — the method Python calls automatically when creating an object, used to set initial attribute values. self refers to the object being created.

13. What is self?

The first parameter of instance methods — the object itself. self.name means "this particular object's name".

14. What is inheritance? Where in your project?

A class receiving the attributes/methods of a parent class. All my models inherit from db.Model — that's what gives them .query, saving, and table mapping. (Template inheritance in Jinja is the same idea for HTML.)

15. Four pillars of OOP?

Encapsulation (data + methods bundled, internals hidden), Inheritance (reuse via parent classes), Polymorphism (same interface, different behaviour — e.g. len() on list or string), Abstraction (expose what, hide how — Event.query.all() hides all SQL).

16. What is a module / import?

A module is any .py file; import loads it so you can use its names. from models import db, User pulls specific names from my own models.py.

17. What does if __name__ == '__main__' do?

True only when the file is run directly (python app.py), not when imported. I use it so the server starts only on direct run.

18. What is an exception? try/except?

An error at runtime (ValueError, KeyError, ZeroDivisionError…). try: runs risky code; matching except: block handles the failure instead of crashing.

19. Common exception types?

ValueError (int("abc")), TypeError (wrong type operation), KeyError (missing dict key — e.g. request.form without that field), IndexError (list[99]), ZeroDivisionError, FileNotFoundError.

20. range(5) gives what?

0,1,2,3,4 — start at 0, stop BEFORE 5. range(1,6) gives 1–5.

21. How do you read the 3rd item of a list?

items[2] — indexing starts at 0. Negative indexes count from the end: items[-1] is the last.

22. What is None?

Python's "no value". Event.query.get(999) returns None when no row exists — that's why I check if not user.

23. int('5') vs str(5)?

Type conversion: int() text→number (form data arrives as text), str() number→text (needed to return numbers from Flask routes).

24. What does len() do? What can it take?

Number of items — works on strings (characters), lists, dicts (keys), tuples. In Jinja the equivalent filter is |length.

25. append() vs extend()?

append adds ONE item (possibly a whole list as one element); extend adds each element of another iterable individually.

26. How to check if something is in a list?

if x in mylist: — also works for keys in dicts and substrings in strings.

27. What is a virtual environment?

An isolated per-project set of installed packages, so projects can use different versions. python -m venv venv, activate with venv\Scripts\activate on Windows. I installed globally for viva reliability — the official checklist permits this.

28. What is pip? What is requirements.txt?

pip is Python's package installer. requirements.txt lists the project's dependencies so pip install -r requirements.txt reproduces them anywhere. Mine has Flask and Flask-SQLAlchemy.

29. What is datetime used for in your project?

datetime.now().strftime('%Y-%m-%d') stamps each booking with today's date as text like 2026-07-07.

30. How would you round up a division (math.ceil)?

import math; math.ceil(2.1) → 3. (Real viva ask: rounding up parking hours.) math.floor rounds down; round() rounds to nearest.

31. What is a global variable?

Defined at file top level, readable inside functions. To REASSIGN one inside a function you'd need global name — better avoided; I pass values as parameters instead.

32. Difference between print and return?

print writes to the terminal for humans; return hands a value back to the caller. A route must RETURN its response — printing it would show it in the terminal, not the browser.

33. What are *args and **kwargs?

Collect extra positional args into a tuple / extra keyword args into a dict, letting functions accept flexible arguments. Flask uses this pattern internally (e.g. url_for('edit_event', event_id=5)).

34. What is string slicing?

s[start:stop] — substring from start up to (not including) stop. "Music Night"[0:3] → "Eve". Works on lists too.

35. How to loop with index?

for i, t in enumerate(events): gives position + item. In Jinja, loop.index inside a for gives 1-based position.

36. What is a docstring / comment?

Comment: # note for humans, ignored by Python. Docstring: a string right under a def/class describing it, readable via help().

37. Write logic to limit a username to 10 characters (live-task classic).

Backend: if len(username) > 10: flash('Username too long'); return redirect(url_for('register')) before creating the user. Frontend: maxlength="10" on the input. Do both, say which is which.

38. Write logic: password must have a capital letter and a special character (live-task classic).

if not any(c.isupper() for c in password) or not any(c in '!@#$%^&*' for c in password): flash('Weak password')... — any() returns True if any element passes.
