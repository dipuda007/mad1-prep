Lesson 1: The Terminal

Goal: stop fearing the black box. You need exactly these commands.

The commands you will actually use

CommandWhat it doesExample
cd foldernamego into a folder ("change directory")cd eventhub-app
cd ..go one folder up
dirlist what is in the current folder
mkdir namecreate a foldermkdir templates
python file.pyrun a Python filepython app.py
pip install nameinstall a Python package from the internetpip install flask
Ctrl+Cstop a running program (e.g. stop the Flask server)
Press the ↑ arrow key in the terminal to repeat previous commands, and Tab to auto-complete file/folder names. These two save enormous time in the viva.

Where am I?

The terminal always shows your current folder in the prompt, e.g. PS C:\projects\eventhub-app>. Flask must always be started from the folder that contains app.py — "it doesn't run" is almost always "you are in the wrong folder".

"What does localhost mean?" — localhost (IP 127.0.0.1) is a special address meaning this same computer. When Flask runs at http://127.0.0.1:5000, the server and the browser are both your own machine; 5000 is the port number Flask listens on by default.
